# Memeflix

## Requirements
- .NET SDK (9.0 recommended)
- Docker
- Optional: Visual Studio 2022 or VS Code + C# extension

### Linux
- Docker-compose

## Quick start
From this folder (project root containing the .csproj):

```bash
dotnet restore
dotnet run
```

Build a composed docker:
```bash
# Windows
docker compose up -d --build

# Linux
docker-compose up -d --build
```
